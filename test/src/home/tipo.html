Tipos de datos

//Declaramos la variable pero no la asignamos
int numero;

//Para asignar un valor se usa el simbolo "=" y el valor que queremos que almacene
numero = 5;

//Tambien se puede declarar y asignar un valor al mismo tiempo
int numero = 5;

Constantes: Se declara una constante con la palabra "const" , también se debe asignar valor. Tras esto la constante queda bloqueada y no se podrá cambiar

Ejemplo:
const double PI = 3.1416;
